Javascript DOM

Dengan HTML DOM, JavaScript dapat mengakses dan mengubah semua elemen dokumen HTML.

Apa itu DOM?

HTML DOM adalah model objek dan antarmuka pemrograman standar untuk HTML. Model ini mendefinisikan:

  • Elemen HTML sebagai objek
  • Properti semua elemen HTML
  • Metode untuk mengakses semua elemen HTML
  • Peristiwa untuk semua elemen HTML

JavaScript - HTML DOM Methods

Metode DOM HTML adalah tindakan yang dapat kalian lakukan (pada Elemen HTML). Properti HTML DOM adalah value (Elemen HTML) yang dapat lu atur atau ubah.

Interface Pemrograman DOM

DOM HTML dapat diakses dengan JavaScript (dan dengan bahasa pemrograman lainnya). Dalam DOM, semua elemen HTML didefinisikan sebagai objek . Antarmuka pemrograman adalah properti dan metode setiap objek. Properti adalah value yang dapat lu peroleh atau atur (seperti mengubah konten elemen HTML). Method adalah tindakan yang dapat lu pada lakukan ( seperti menambah atau menghapus elemen HTML). 


  document.getElementById("demo").innerHTML = "Hello World!";
